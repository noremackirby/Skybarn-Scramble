What Makes It Work

Website and game in one page

The page is a normal website layout wrapped around a playable canvas game, so you can open one file in Chrome and get both the presentation and the gameplay.

Original Design

Built from scratch

The visuals, course, scoring, obstacle types, and interaction flow are all original to avoid copying another game's protected content.
